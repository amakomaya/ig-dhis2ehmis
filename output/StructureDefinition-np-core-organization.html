Resource Profile: NP Core Organization Profile

Official URL: https://fhir.hmis.gov.np/StructureDefinition/np-core-organization Version: 0.0.1-ballot
Active as of 2025-08-20 Computable Name: NPCoreOrganizationProfile

Copyright/Legal: Used by permission of HL7 International, all rights reserved Creative Commons License

The NP Core Organization Profile inherits from the FHIR Organization resource. It defines the minimum expectations for representing organizations (e.g., health facilities, departments, NGOs, government agencies) within the Nepal health system. It specifies required elements, identifiers, and standard terminologies.
