Resource Profile: NP Core Condition Profile

Official URL: https://fhir.hmis.gov.np/StructureDefinition/np-core-condition Version: 0.0.1-ballot
Active as of 2025-08-20 Computable Name: NPCoreConditionProfile

Copyright/Legal: Used by permission of HL7 International, all rights reserved Creative Commons License

The NP Core Condition Profile inherits from the FHIR Condition resource. This profile defines minimum expectations for capturing diagnoses, risk conditions, and program-specific conditions in the Nepal health system. It ensures consistent data capture, searchability, and interoperability across national health programs.

Usages:

  • This Profile is not used by any profiles in this Implementation Guide
