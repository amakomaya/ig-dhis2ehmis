Resource Profile: NP Core Medication Dispense Profile

Official URL: https://fhir.hmis.gov.np/StructureDefinition/np-core-medication-dispense Version: 0.0.1-ballot
Active as of 2025-09-07 Computable Name: NPCoreMedicationDispenseProfile

Copyright/Legal: Used by permission of HL7 International, all rights reserved Creative Commons License

The NP Core MedicationDispense Profile inherits from the FHIR MedicationDispense resource; refer to it for scope and usage definitions. This profile sets minimum expectations for the MedicationDispense resource to record, search, and fetch information about medications dispensed to a patient within the Nepal health system. It specifies which core elements, extensions, vocabularies, and value sets SHALL be present and constrains how the elements are used.

Usages:

  • Examples for this Profile: MedicationDispense/medication-dispense-albendazole, MedicationDispense/medication-dispense-calcium and MedicationDispense/medication-dispense-iron

You can also check for usages in the FHIR IG Statistics
